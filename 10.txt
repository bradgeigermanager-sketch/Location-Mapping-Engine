🧭 3. Algorithms Used (Complete Set)

A. Dimensions 1–3 (3D Space)

Euclidean
\[
D{3D} = \sqrt{(xA - xB)^2 + (yA - yB)^2 + (zA - z_B)^2}
\]

Manhattan
\[
M{3D} = |xA - xB| + |yA - yB| + |zA - z_B|
\]

Chebyshev
\[
C{3D} = \max(|xA - xB|, |yA - yB|, |zA - z_B|)
\]

Direction vector
\[
\vec{v} = (xB - xA,\; yB - yA,\; zB - zA)
\]

---

B. Dimension 4 (Fixed Brane Separation)

\[
D4 = |d^{(A)}4 - d^{(B)}_4|
\]

This is time‑invariant.

---

C. Dimensions 5+ (Brane Geometry)

Per‑dimension static distance
\[
\Delta dk = |b^{(A)}k - b^{(B)}_k|
\]

Per‑dimension static angle
\[
\Delta \theta_k = \arccos\left(
\frac{b^{(A)}k \cdot b^{(B)}k}{|b^{(A)}k|\,|b^{(B)}k|}
\right)
\]

Per‑dimension distance rate
\[
\dot{d}k = \frac{dk(t2) - dk(t1)}{t2 - t_1}
\]

Per‑dimension angle rate
\[
\dot{\theta}k = \frac{\thetak(t2) - \thetak(t1)}{t2 - t_1}
\]

---

D. Composite Static Brane Distance

\[
D{\text{static}} = \sqrt{\sum{k=5}^{n} (\Delta d_k)^2}
\]

---

E. Composite Dynamic Divergence

\[
R_{\text{dynamic}} = \sqrt{
\sum{k=5}^{n} \left( \dot{d}k^2 + \beta \dot{\theta}_k^2 \right)
}
\]

---

F. Total Distance (Metric Profile Driven)

\[
D_{\text{total}} =
\sqrt{
\gamma4 D4^2 +
\gammas D{\text{static}}^2 +
\gammaa \sum{k=5}^{n} (\Delta \theta_k)^2 +
\gammad R{\text{dynamic}}^2
}
\]

Where all \( \gamma \) and \( \beta \) come from the metric profile.
